index.html 파일은 메인 페이지 역할을 함 어떻게 자바랑 html 연결되나요? -> web.xml 파일 확인

	이름, 나이를 입력받아 서버로 제출
	***** html 지식 *****
	1) input 태그 : 값을 입력할 수 있는 양식을 제공하는 태그
			name 속성					: input 태그 값(value)이 서버로 제출될 때
					   				  key 역할을 하는 속성
			value 속성 또는 작성된 값		: 서버로 제출되는 값

	2) form 태그 : 내부에 작성된 입력 값 (input 태그)를 지정된 경로로 제출
			지정된 경로 : 다른 html 파일의 경로, 서버(Java servlet)에 요청한 주소
			action 속성 : 제출할 경로 또는 주소를 작성하는 속성
			method 속성 : 제출 방식을 지정하는 속성 (기본 값 : GET)
